# AgendaSimple
Contact Agenda for Android for the Mobile Development Master Degree Android Development Course in UPM. Available in ESP and ENG.

App functionality:

  - Export all contacts and their information to the external storage via cnt file.
  - Import all contacts from the backup in the external storage.
  - Import all contacts from the contact list in your own device.
  - Call contacts.
  - Search contacts.
  - Mark as favourites.
  - Access to WhatsApp in order to send directly a message to the desired contact.
  - Access to the Mail Management App at will to send emails to the desired contact.
  - Add photos to the contacts from the gallery of the device.
  - Add, modify and cancel dates with contacts via Google Calendar.
